# Basic Java Programs

This repository contains a collection of **basic Java programs** created to practice
core Java concepts such as conditions, loops, switch cases, and basic logic building.
These programs are beginner-friendly and helpful for students who are learning Java
and problem-solving fundamentals.

---

## Programs Included

- ArmstrongNumbers.java – Checks whether a number is an Armstrong number
- Calculator_IfElse.java – Performs basic arithmetic operations using if-else
- Calculator_Switch.java – Performs basic arithmetic operations using switch case
- CountOccurrence.java – Counts the occurrence of a digit in a number
- EvenOrOdd.java – Checks whether a number is even or odd
- Fibonacci.java – Prints the Fibonacci series
- Greetings.java – Displays greeting messages
- HCFandLCM.java – Finds the HCF and LCM of two numbers
- LargestNumBtw2.java – Finds the largest number between two numbers
- LeapOrNot.java – Checks whether a year is a leap year
- Months.java – Displays holidays based on the month using switch case
- MultiplicationTable.java – Prints the multiplication table of a number
- PalindromeOrNot.java – Checks whether a number is a palindrome
- RupeeToUSD.java - Converts ₹ to $
- SimpleInterest - SI = (P*T*R)/100
- SumOf2No - Adding 2 numbers
- SumUntilXInput - Keep adding elements untill the user presses X.
- Temperature - Converts Celsius to Fahrenheit

## How to Run the Programs

1. Clone the repository:
   ```bash
   git clone <your-github-repository-link>
2. Navigate to the project folder:
   cd Basic
3. Compile the Java file:
   javac FileName.java
4. Run the program:
   java FileName
